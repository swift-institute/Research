## [COLLAB-014] (promoted 2026-07-05)

**Rule**: [COLLAB-014] ChatGPT Is a Challenge Generator, Not an Authority.

**Provenance**: memory `feedback_collab_chatgpt_not_gospel` (2026-06-10). Promoted by the 2026-07-05 memory-drain (D1 cluster: supervise / handoff / collaborative-discussion / reflect-session / quick-commit-and-push-all).

**Origin**: In `/collaborative-discussion` rounds, the principal (2026-06-10) rated the current Claude model as the stronger analyst; ChatGPT's value is producing interesting adversarial challenges and different-training perspectives, not authoritative conclusions. External-reviewer framing creates deference pressure, and deferring to a weaker analysis degrades converged plans.

**Reasoning**: the round protocol ([COLLAB-002]) presents Claude and ChatGPT symmetrically, which can read as parity of authority. [COLLAB-013] already requires pushback-with-rationale; [COLLAB-014] adds the epistemic stance behind it — Claude's analysis is primary, ChatGPT's rounds are mined for genuinely new failure modes / prior art / counterexamples, and any adopted claim is independently verified (source, probes, docs) before it moves Claude's Position. Status advances on resolved substance, never on the external reviewer's authority. Composes with the CLAUDE.md Collaboration Protocol "challenge implementations / do not rubber-stamp" directive.
